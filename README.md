It's easy to spin up 100 agents in Codex, but merging them back together is hard. BranchMaster was built to fix that.

[BranchMaster](https://branchmaster.dev) is a desktop app that lets you manage your parallel agents and branches in one place. Branch, commit, merge, and push, without fumbling with Codex or Cursor.
