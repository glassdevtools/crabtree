# BranchMaster Breakdown Website

Static UI state breakdown for the BranchMaster product surfaces.

Open `index.html` directly in a browser, or run from the repo root:

```bash
npm run dev:breakdown-website
npm run build:breakdown-website
npm run test:breakdown-website
```

The breakdown data is split by product part:

- `data/product-surface.js`
- `data/website.js`
- `data/desktop-app.js`
